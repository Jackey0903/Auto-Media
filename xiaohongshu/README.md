# Auto-Media (全自动小红书内容矩阵助手) 🚀

> **告别 AI 味，打造真正有灵魂的自动化自媒体账号。**

这是一个专为小红书设计的全自动内容生成与发布系统。它不仅仅是一个简单的“发帖机器人”，而是一个集成了**深度搜索、仿人写作、智能审图、自动化部署**的完整解决方案。

无论你是想做热点新闻搬运、行业科普，还是论文深度解读，Auto-Media 都能帮你全天候自动运行，从选题到发布，无需人工干预。

![Web UI](xiaohongshu_web_ui_1770774117525.png)

## 🌟 为什么选择它？

### 1. 拒绝“AI 塑料感” ✍️
我们厌倦了“首先、其次、综上所述”的僵硬 AI 文风。
*   **真实感写作**：内置精心调教的 Prompt，模拟真实博主的口语化表达，有情绪、有观点。
*   **深度内容**：不是简单的拼凑，而是基于 Tavily/Jina 全网搜索的深度整合，确保信息准确、硬核。

### 2. 配图不再是痛点 🖼️
文章写得好，配图更要吸睛。
*   **海量备选**：系统会自动搜索 **10+ 张**相关图片作为备选池。
*   **智能筛选**：自动过滤模糊、尺寸不合规、防盗链的图片，确保最终选出的 **5-7 张** 图片张张可用。
*   **自动重试**：下载失败？系统会自动替补，绝不开天窗。

### 3. 服务器级的稳定性 🛡️
专为长期运行设计，不再担心报错挂机。
*   **全自动调度**：内置调度器，支持每小时/每天自动运行，无需人工守候。
*   **容器化部署**：提供 Docker 支持，一键在服务器上启动，环境零配置。
*   **智能纠错**：
    *   标题超长？自动缩短。
    *   内容太长？自动总结。
    *   偶尔 500 错误？自动重试。

## 🚀 极速上手

### 方式一：Docker 一键部署 (推荐) 🐳

适合在服务器或本地长期运行。无需安装 Python 环境，省心省力。

```bash
# 1. 克隆代码
git clone https://github.com/Jackey0903/Auto-Media.git
cd Auto-Media

# 2. 配置 (仅需一次)
# 将你的 cookies.json 和 app_config.json 放入对应目录 (详见 DEPLOY.md)

# 3. 启动！
docker compose up -d --build
```

系统将立即开始工作，默认每小时自动寻找热点并发布。

> 详细部署教程请阅读：**[部署指南 (DEPLOY.md)](./DEPLOY.md)**

### 方式二：本地源码运行 💻

适合开发调试或只需偶尔生成一篇。

1.  **安装依赖**：
    ```bash
    pip install -r requirements.txt
    cd xiaohongshu-mcp-darwin-arm64 && npm install
    ```
2.  **启动 Web UI**：
    ```bash
    python app.py
    ```
3.  浏览器访问 `http://localhost:8080`，可视化操作。
